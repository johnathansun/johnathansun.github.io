# Immaculate

The personal website of Johnathan Sun. Find at [jsun.one](https://jsun.one).

## Colophon

This site is built with [Jekyll](https://jekyllrb.com), stored on [Github](https://github.com/), and served with [Netlify](https://www.netlify.com). The visuals are adapted from the [Immaculate](https://github.com/siawyoung/immaculate) theme and by extension, [Tufte CSS](https://github.com/edwardtufte/tufte-css). The typeface is [ET Book](https://edwardtufte.github.io/et-book/). Many thanks to these developers for their beautiful work.
